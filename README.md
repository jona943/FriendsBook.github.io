# Librería "Friend Book" - Página Web

Este es el proyecto final del curso de desarrollo Frontend, que consiste en una página web estática para una librería ficticia llamada "Friend Book". La página presenta información sobre la librería, sus servicios, libros recomendados y un formulario de contacto.

## Caracteristicas

La página web está organizada en las siguientes secciones:

-   **Nosotros:** Una breve introducción sobre la historia y misión de la librería.
-   **Servicios:** Lista de los servicios ofrecidos, como venta de libros, club de lectura y cafetería.
-   **Recomendados:** Una galería con las portadas de libros recomendados, cada uno con su título y una breve descripción.
-   **Solicitud de libro:** Un formulario de contacto para que los usuarios puedan solicitar libros que no encuentren en la tienda.

## Tecnologias Utilizadas

-   **HTML5:** Para la estructura y el contenido semántico de la página.
-   **CSS3:** Para el diseño, la maquetación y la estética visual, incluyendo el uso de Flexbox para la alineación de elementos.

## Como Visualizar el Proyecto

Para ver la página web, simplemente abre el archivo `index.html` en tu navegador web preferido.

1.  Clona o descarga este repositorio en tu máquina local.
2.  Navega a la carpeta del proyecto.
3.  Haz doble clic en el archivo `index.html` o arrástralo a la ventana de tu navegador.
